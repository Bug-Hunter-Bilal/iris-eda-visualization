#  Iris Dataset - Exploratory Data Analysis

##  Project Overview
This project performs Exploratory Data Analysis (EDA) on the Iris dataset using Python.

The objective is to:
- Understand dataset structure
- Generate summary statistics
- Visualize relationships between features
- Identify outliers

---

##  Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

##  Dataset
The Iris dataset contains 150 samples with 4 features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target variable:
- Species (Setosa, Versicolor, Virginica)

---

##  Visualizations Performed
✔ Scatter Plot  
✔ Histograms  
✔ Box Plots  
✔ Correlation Heatmap  

---

##  Key Observations
- Petal features show strong correlation.
- Setosa is clearly separable from other species.
- Minimal outliers detected.
